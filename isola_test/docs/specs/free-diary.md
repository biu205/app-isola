# 自由日記功能規格

## 基本資訊
- **功能名稱**：自由日記模式（Free Diary Mode）
- **版本**：v1.1
- **狀態**：實作中
- **優先度**：中
- **建立日期**：2026-05-31

## 目的
為使用者提供不受結構限制的日記記錄選項，支援自由撰寫、多媒體輸入，滿足希望隨時記錄生活片段的使用者需求。作為 AI 聊天日記與隨機問答日記的補充選項。

## 範圍

### In-Scope（包含）
- 自由文字輸入（無字數限制）
- 多媒體支援（圖片、影片）
- 日期時間標記
- 草稿自動保存
- 日記編輯與刪除
- 富文本格式化（粗體、斜體、顏色標記等）

### Out-of-Scope（不包含）
- 日記版本控制（階段二）
- 協作編輯（階段二）
- 內容導出（階段二）

## 使用者故事

### 故事 1：即興記錄
**As a** 忙碌的上班族  
**I want to** 隨時能快速記錄靈感或發生的事  
**So that** 我不會遺漏重要的生活片段

### 故事 2：詳細敘述
**As a** 喜歡詳細記錄的使用者  
**I want to** 能自由撰寫長篇幅的日記，不受題目限制  
**So that** 我能詳盡表達我的想法與感受

### 故事 3：圖文並茂
**As a** 視覺化思考的使用者  
**I want to** 能在日記中插入圖片與影片  
**So that** 我的日記更生動且更好回顧

## 功能流程

```
1. 使用者點擊「自由日記」或「+」新建按鈕
2. 進入空白編輯介面
3. 系統自動填入當前日期與時間
4. 使用者可選擇：
   - 輸入標題（可選）
   - 編寫正文
   - 插入媒體（圖片/影片）
   - 添加標籤（可選）
5. 草稿自動保存（每 30 秒或每次編輯後）
6. 使用者完成編輯後點擊「保存」或「完成」
7. 日記存儲至本地與雲端
8. 返回日記列表或首頁
```

## 技術實現

### 資料結構
```swift
struct FreeDiary {
    let id: UUID
    var title: String?
    var content: String
    var date: Date
    var mediaAttachments: [MediaAttachment]
    var tags: [String]
    var lastEditedAt: Date
    var isSynced: Bool
}

struct MediaAttachment {
    let id: UUID
    let type: MediaType // image, video
    let localPath: String
    let uploadedURL: String? // Firebase Storage
    let duration: Double? // For video
}

enum MediaType {
    case image
    case video
}
```

### 自動保存機制
- 事件觸發：每 30 秒或使用者停止編輯 5 秒後
- 存儲位置：SwiftData（本地快取）
- 同步策略：後台隊列同步至 Firestore
- 衝突解決：最新修改時間戳記優先

### 媒體處理
- 圖片：自動壓縮至最大 5MB，支援 HEIC/JPG/PNG
- 影片：自動壓縮至最大 50MB，支援 MP4/MOV
- 上傳位置：Firebase Storage
- CDN 快速存取

## 非功能需求

- **效能**：編輯介面加載 < 500ms；自動保存 < 2 秒
- **可用性**：UI 簡潔，新使用者無教學也能使用
- **隱私**：所有媒體上傳前加密；使用者可刪除上傳的媒體
- **離線支援**：編輯的內容在無網路時本地保存，恢復連接後同步

## 文字編輯能力

### 支援的格式化

| 格式 | 實現方式 |
|------|--------|
| 粗體 | **文字** 或 Cmd+B |
| 斜體 | *文字* 或 Cmd+I |
| 標題 | # 標題 |
| 項目符號 | - 項目 |
| 連結 | [文字](URL) |
| 標籤 | #標籤名 |

### 拼寫檢查
- 支援繁體中文與英文拼寫檢查
- 使用者可自訂字典

## 媒體限制

| 類型 | 上限 | 格式 |
|------|------|------|
| 圖片 | 5 個 / 篇 | HEIC, JPG, PNG |
| 影片 | 1 個 / 篇 | MP4, MOV |
| 檔案大小 | 5MB (圖) / 50MB (影) | — |

## 相依項

- Firebase Storage 配額充足
- SwiftData 本地存儲空間 ≥ 200MB
- 相機與相簿權限授予

## 數據儲存

### SwiftData（本地）
- 日記內容（文字）
- 媒體附件本地路徑
- 同步狀態

### Firestore（雲端）
- 日記元數據（標題、日期、標籤）
- 媒體連結（Firebase Storage URL）

### Firebase Storage
- 實際媒體檔案
