# 情緒日曆視覺化功能規格

## 基本資訊
- **功能名稱**：情緒日曆視覺化（Emotion Calendar Visualization）
- **版本**：v1.1
- **狀態**：實作中
- **優先度**：中
- **建立日期**：2026-05-31

## 目的
以月曆形式視覺化展示每日的情緒狀態，幫助使用者一眼掌握整月的情緒週期與趨勢，強化「我在進步」的正面反饋，並提供長期情緒追蹤的視覺證據。

## 範圍

### In-Scope（包含）
- 月曆視圖展示
- 情緒圖示與顏色編碼
- 月份切換與導航
- 日期點擊查看詳情
- 情緒統計與趨勢分析
- 匯出/截圖功能

### Out-of-Scope（不包含）
- 年度檢視（階段二）
- 情緒對比與預測（階段二）
- 社交分享（階段二）

## 使用者故事

### 故事 1：掌握情緒週期
**As a** 想了解自己情緒模式的使用者  
**I want to** 一眼看到整個月的情緒狀態分布  
**So that** 我能發現情緒何時最低落、何時最開心

### 故事 2：看見進步
**As a** 需要正面鼓勵的使用者  
**I want to** 看到這個月有很多開心的日子（用顏色視覺化）  
**So that** 我感受到自己在變好

### 故事 3：追蹤特定日期
**As a** 想回顧特定日期的使用者  
**I want to** 點擊月曆上的某一天查看那天的日記與情緒詳情  
**So that** 我能快速回到想起的特定時刻

## 功能流程

```
1. 使用者進入「情緒日曆」頁面
2. 系統加載當月的情緒數據
3. 以月曆形式展示，每個日期格子有：
   - 日期編號
   - 情緒圖示（😊 😐 😢 等）
   - 顏色背景（綠/黃/紅 代表不同情緒）
4. 使用者可向左/右切換月份
5. 選擇上月/下月後日曆更新
6. 點擊某一天可查看詳情
7. 詳情頁顯示：
   - 當天記錄的所有日記
   - 生理數據（若有）
   - 該日情緒的主要原因（AI 摘要）
8. 使用者可返回月曆或分享該天
```

// 情緒推導：

## 非功能需求

- **效能**：
  - 月曆加載 < 500ms
  - 月份切換 < 300ms
  - 詳情頁加載 < 1 秒
  
- **儲存**：月曆數據 < 2MB（含圖片）
- **動畫**：月份切換、情緒變化動畫流暢

- **功能**：使用者可截圖或導出月曆
- **格式**：PNG 或 PDF
- **內容**：月曆視圖 + 統計資訊（若勾選）
- **隱私**：使用者可選擇是否包含詳細資訊
